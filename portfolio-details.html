Project description

One of the essential processes in data engineering is the Extract, Transform, and load process (ETL). This process helps us to move data from one system to another, reformat it and even clean it up when necessary. In this project, I used Python to extract data from a CSV file and load it into a MySQL database

Usually, every ETL process starts by extracting data from the source. I carried out the project using pyspark. I read the CSV data using the pyspark, performed some data transformation, and stored it in the MySQL database

To achieve this, I started creating a spark session and connected it to the MySQL database. I then created a cursor object and executed the SQL commands to create the table from my CSV data. I then iterated over the rows in my csv file extracting the data from each row and inserting it into mysql table using an SQL insert command

Project description

ETL is a process that I use to extract, transform, and load data from one system to another. Pyspark is a big data processing engine that can process huge datasets quickly. This project explored a simple ETL process that extracts data from a Postgresql, transform it using pyspark, and load it to the MySQL database

To connect the PostgreSQL to the pyspark mysql-connector library is used. The process I took in this includes;

Extracting data from Postgresql. Data extraction is the first process in every ETL process.

Data transformation is the second step. Data needs to be transformed in order to transform it into the format that can be loaded to the target MySQL database. Several processes are can be carried out in this stage. Some of the processes carried out in this step include; data type conversion, data cleansing, and data aggregation. In this process, I carried out some data transformation and grouped the data into a salary table in relation to the salary. The grouped data was filtered to employees with above 150k salary.

The last stage is the data loading stage. Here I loaded data into the MySQL database. To connect the pyspark to the mysql mysql connector library is used. Finally, I loaded the grouped table to MySQL database in the database called users and orders table
